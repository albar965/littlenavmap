This folder contains example configuration files for the map and user interface styles.
Copy modified files to `C:\Users\YOURUSERNAME\AppData\Roaming\ABarthel` to override default styles and settings.

See the online manual for more information how to use these files:
https://www.littlenavmap.org/manuals/littlenavmap/release/latest/en/CUSTOMIZE.html

----

The file `little_navmap_alternative_darkstyle.ini` contains an alternative night style by merspieler.
